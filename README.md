# Vector

Voice-guided cycling beacon — навигация «по часам» вместо градусов и стрелок.

Сейчас в репо лежит **HTML-прототип** из дизайн-хэндоффа. JSX компилируется Babel'ом прямо в браузере — ничего собирать не нужно, можно открыть `index.html` локально или задеплоить как статику.

Полная спецификация и план перевода на Vite+React+PWA — в [DESIGN-HANDOFF.md](./DESIGN-HANDOFF.md).

## Локальный запуск

```bash
# любой статический сервер, например:
npx serve .
# или:
python -m http.server 8000
```

Открыть `http://localhost:8000`.

## Деплой

Подключён к Netlify, билд тривиальный — Netlify раздаёт корень репо. См. `netlify.toml`.
