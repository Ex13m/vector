# История изменений

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/), нумерация — [Semantic Versioning](https://semver.org/lang/ru/).

## [0.2.0] — 2026-05-05

Полная пересборка прототипа в production-стек. Из дизайн-референса — в рабочий PWA.

### Добавлено

- **Vite 5 + React 18 + TypeScript** вместо Babel-в-браузере. Бандл собирается, минифицируется, code-splittится.
- **Service Worker** через `vite-plugin-pwa` (Workbox): precache app shell, runtime-кэш для шрифтов и тайлов карт, навигационный fallback.
- **Тост обновления**: `useRegisterSW({ onNeedRefresh })` → пользователь видит «Доступно обновление · ОБНОВИТЬ», по клику SW активируется и страница перезагружается.
- **Установка PWA**: баннер «Установить Vector» по событию `beforeinstallprompt` (Android) и инструкция «Поделиться → На экран Домой» (iOS) с откладыванием на 7 дней.
- **MapLibre GL JS** на всех трёх экранах. Стили: OSM Standard, Esri World Imagery (спутник), OpenTopoMap, Waymarked Trails. Растровые источники с тёмным `raster-saturation` и приглушённой контрастностью.
- **Реальная геолокация** (`Geolocation.watchPosition`, `enableHighAccuracy`) — позиция, скорость, аппаратный heading, точность.
- **Ориентация устройства** (`DeviceOrientationEvent.absolute` или `webkitCompassHeading` для iOS) с экспоненциальным сглаживанием 0.25 и запросом разрешения для iOS 13+.
- **Клавишный голос** (`SpeechSynthesis`): фраза «Цель на N часов, X км» каждые `intervalSec` (по умолчанию 15 мин). Кнопка sayNow в HUD. Можно выбрать конкретный голос в настройках.
- **Хаптика** (`navigator.vibrate`): короткий импульс при смене «часа», тройной паттерн при попадании в 12 часов.
- **Офлайн-кэш карты**: расчёт BBox + zoom levels, последовательная закачка тайлов в IndexedDB через `idb-keyval` с конкурентностью 6, прогресс-бар, отмена.
- **Сохранение** избранных точек (до 50) и поездок (до 100) в IndexedDB.
- **Локализация RU/EN** через `i18n.ts` со словарями и подстановкой переменных. Авто-выбор по `navigator.language`.
- **Auto-hide chrome**: top bar и floating-кнопки тают до opacity 0.18 через 5 секунд бездействия. Тап по карте — снова видны.
- **Long-press peek**: 380 ms на mini-циферблате → fullscreen ClockDial 280×280 с цифрами часов, авто-закрытие через 2.2 с.
- **Near-target glow** (`<500 м`): нижний HUD меняет фон/glow/акцентный цвет с оранжевого на зелёный, transition 400 ms.
- **Arrived overlay**: при подходе ближе 50 м — экран финиша с результатами, инпутом названия и сохранением в IndexedDB.
- **Settings sheet**: интервал голоса (slider 0–1800 step 60), единицы (метрические/мили), хаптика, язык, выбор голоса.
- **Reverse-курс**: режим «вернуться назад» — цель становится стартом, цель ‒ исходная точка пользователя.
- **CHANGELOG.md** и обновлённый **README.md** с инструкцией для разработчика и пользователя.

### Изменено

- Прототип целиком перенесён в папку `prototype/` как референс.
- Иконки PWA — отдельные файлы `any` и `maskable` в `public/`.
- `netlify.toml`: build = `npm run build`, publish = `dist`, SPA-fallback, корректные `Cache-Control` для `sw.js` / `manifest`.

### Удалено

- Babel standalone и React UMD из `index.html` — теперь нормальный ESM-бандл.
- `android-frame.jsx` и `tweaks-panel.jsx` — это были инструменты прототипа.
- Симулятор движения с random-drift bearing — заменён реальным GPS.

## [0.1.0] — 2026-05-05

Первый деплой — статический HTML-прототип из дизайн-хэндоффа.

### Добавлено

- HTML-прототип `Vector.html` с JSX через Babel-в-браузере.
- Три экрана: Pick / Cache / Ride.
- Дебаг-панель Tweaks (язык, симуляция, экран, слой).
- Иконки PWA, manifest, простой service worker (precache + cache-first).
- Тост «Доступно обновление».
- `netlify.toml` для статической раздачи прототипа.
- README с описанием прототипа.
- Деплой на Netlify через GitHub.

### Известные ограничения

- Геолокация **не использовалась** — позиция симулировалась случайным дрейфом.
- Карта была **нарисованной декорацией**, без реальных тайлов.
- Голос — только в спецификации.
- Поездки в памяти вкладки, без сохранения.
